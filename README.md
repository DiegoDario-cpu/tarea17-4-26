# tarea17-4-26
Este proyecto consiste en el desarrollo de dos clases de utilidad que implementan 10 funcionalidades específicas utilizando programación funcional en Java.

Autor: Diego Dario Figueroa Mejia

Descrpcion de Funciones:
-CalculadoraNumerica
1.  **El Multiplicador:** Escala todos los elementos de un `ArrayList<Integer>` por un factor dado usando `replaceAll`.
2.  **Cuadrados Únicos:** Filtra números pares, calcula su cuadrado y los devuelve en un `HashSet` para evitar duplicados.

ProcesadorCadenas
3.  **El Filtro Selectivo:** Elimina cadenas que inician con una letra específica o tienen una longitud menor a N.
4.  **Conversor de Mayúsculas:** Transforma una lista de palabras a mayúsculas utilizando `.map()`.
5.  **Mapa de Longitudes:** Crea un `HashMap` donde la clave es la palabra y el valor es su longitud.
6.  **Modificador de Inventario:** Imprime un catálogo de productos aplicando un 10% de descuento visual mediante `Bi-Consumer`.
7.  **Contador de Frecuencias:** Construye un mapa de frecuencias de palabras usando el método `merge`.
8.  **Clasificador de Palabras:** Filtra un mapa de frecuencias para obtener solo las palabras que superan un umbral N.
9.  **Deduplicación de Palabras:** Procesa una frase, filtra palabras cortas, las convierte a minúsculas y elimina repetidas.
10. **El Tope de Frecuencias:** Limita los valores de un mapa de frecuencias a un máximo N usando `replaceAll`.
